# DeepGELOB 论文材料说明

本仓库暂不公开此前的论文原稿或投稿版 PDF。

原因包括：

1. 后续标签审计发现，早期平滑收益目标含有可从历史输入恢复的成分。原稿中的实验结论需要用严格未来收益标签重新验证；
2. 修正后的实验尚需补齐跨股票、跨月份、完整训练轮次和成本评价；
3. 论文由多人合作完成，公开署名稿、源码或投稿状态前需取得共同作者同意；
4. 若稿件仍处于匿名评审流程，还需确认会议对预印本、公开代码和身份线索的规定；
5. 原始行情数据存在授权边界，不能随论文材料公开。

公开仓库只保留独立重写的模型、标签与评价核心，以及不使用真实行情的合成数据演示。完成修正实验、共同作者确认和评审合规检查后，再决定是否发布修订稿。当前请将该项目描述为“研究中的方法与评测框架”，不要表述为已录用论文或可交易策略。
