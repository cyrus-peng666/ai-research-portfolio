# 数据与结果公开说明

## 不随仓库发布的内容

- 券商、交易所、客户或课题组提供的原始数据；
- 逐笔委托、订单 ID、应用序列号、真实盘口快照和模型预测明细；
- 训练权重、服务器路径、运行日志及未获授权的第三方论文；
- 能够反推出非公开数据口径的逐日结果。

## 仓库提供的替代材料

- 明确的数据 schema；
- 不含真实证券和日期的合成 LOB 生成器；
- 可在 CPU 上运行的模型前向与梯度测试；
- 标签、时间切分和评测口径的单元测试；
- 仅用于展示流程的 deterministic Research Agent。

这些材料用于核验研究方法和工程能力，不应被理解为公开了完整生产数据链路，也不构成任何投资或交易建议。
